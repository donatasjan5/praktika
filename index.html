Git

Git'as yra populiariausia kodo versijavimo programa.

Git Komandos

  • git init - sukuria git'o repozitoriją
  • git add - prideda nurodytus failus prie commit'ui paruoštų failų (staged)
  • git status - nurodo informaciją apie tavo lokalios git'o repozitorijos statusą
  • git commit -m "žinutė" - sukuria kodo versiją iš stage'intų failų ir prideda žinutę apie pakitimus (žinutę bandyti išlaikyti ne ilgesnę negu 50 simbolių)
  • git push - iš lokalios repozitorijos commit'ų nusiuntimas į išorinę repozitoriją
  • git pull - iš išorinės repozitorijos commit'ų atsisiuntimas į lokalią repozitoriją
  • git remote add origin https://... - išorinės repozitorijos prijungimas prie lokalios repozitorijos
  • git log - commit'ų sąrašas
  • git checkout branchName|commitID - pereiti i kita kodo saka arba versija
  • branch naujosSakosPavadinimas - sukuria nauja saka

GitHub

GitHub'as yra viena populiariausių svetainių, kuri skirta talpinti ir dalintis versijuojamu kodu.
